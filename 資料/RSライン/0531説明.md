結論：**RSラインは「その銘柄が基準より強いか弱いか」を見る線**です。
価格チャートだけでは分からない、**市場全体やベンチマークに対する相対的な強さ**が分かります。

## RSラインとは

RSは **Relative Strength** の略です。
ここでいうRSラインは、RSIとは別物です。

基本式はこれです。

```text
RSライン = 対象銘柄の価格 ÷ ベンチマークの価格
```

例：

```text
NVDA ÷ XYZ100
AAPL ÷ SP500
SOL ÷ BTC
ETH ÷ BTC
```

Botでは、数学的にはこの形の方が扱いやすいです。

```text
logRS = log(対象銘柄価格) - log(ベンチマーク価格)
```

意味は同じで、**対象がベンチマークより強いか**を見ます。

---

## 何が分かるのか

### 1. 市場全体より強いか

価格が上がっていても、市場全体も同じくらい上がっているなら、特別強いとは言えません。

例：

```text
NVDA +3%
XYZ100 +3%
```

この場合、NVDAは上がっていますが、相対的には普通です。

一方で、

```text
NVDA +3%
XYZ100 +1%
```

なら、NVDAはベンチマークより強い。
RSラインは上がります。

---

### 2. 価格より先に強さが出ているか

価格チャートではまだ高値を抜けていなくても、RSラインが先に新高値になることがあります。

これは、

```text
価格はまだ目立っていない
でも、ベンチマーク比では強くなっている
```

という状態です。

候補銘柄を早めに見つける用途に使えます。

---

### 3. 弱い銘柄を避けられる

価格が少し上がっていても、ベンチマークより弱いなら、あまり良い候補ではありません。

例：

```text
AAPL +1%
XYZ100 +3%
```

AAPLは上がっていますが、ベンチマークには負けています。
RSラインは下がります。

この場合、

```text
上がっているように見えるが、実は弱い
```

と判断できます。

---

## どう使うのか

### 使い方1：候補銘柄を絞る

一番基本的な使い方です。

```text
全銘柄
↓
出来高が増えている
↓
価格も動いている
↓
RSラインが上昇している
↓
RSラインが新高値
↓
監視候補に入れる
```

重要なのは、**RS新高値＝即買いではない**ことです。

正しくは、

```text
RS新高値 = 監視候補に昇格
```

です。

---

### 使い方2：ランキングする

複数銘柄を比べるときに使えます。

例：

```text
NVDA / XYZ100
AAPL / XYZ100
MSFT / XYZ100
TSLA / XYZ100
```

それぞれのRSを見て、

```text
どれが指数より一番強いか
```

を順位付けできます。

これは、Botの **ユニバース選定** に向いています。

---

### 使い方3：市場レジームを見る

個別銘柄だけでなく、指数同士にも使えます。

例：

```text
XYZ100 ÷ SP500
```

これが上がっているなら、

```text
成長株・テック株が大型株全体より強い
```

という意味になります。

つまり、

```text
リスクオン寄り
```

と判断しやすい。

逆に下がっているなら、

```text
成長株・テック株が弱い
リスクオフ寄り
```

と見られます。

---

## 具体例

### 例1：SOL / BTC

```text
SOL価格 ÷ BTC価格
```

これが上がるなら、

```text
SOLがBTCより強い
```

という意味です。

SOLの価格が横ばいでも、BTCが下がっていればRSは上がることがあります。
つまり、**価格だけでは見えない相対的な強さ**が見えます。

---

### 例2：NVDA / XYZ100

```text
NVDA価格 ÷ XYZ100
```

これが新高値なら、

```text
NVDAが成長株指数全体より強い
```

という意味です。

ただし、単にNVDAが荒く動いているだけの可能性もあります。
だから、RSだけでなく出来高、スプレッド、ボラ、手数料も見る必要があります。

---

## RSラインで分かること・分からないこと

| 項目               |        分かるか |
| ---------------- | ----------: |
| ベンチマークより強いか      |         分かる |
| 市場全体につられて上がっただけか |     ある程度分かる |
| 強い銘柄のランキング       |         分かる |
| 価格に先行する強さ        | 見つけられることがある |
| 将来必ず上がるか         |       分からない |
| どこで買えばよいか        |    直接は分からない |
| 損切り位置            |    直接は分からない |
| 約定コストに勝てるか       | RSだけでは分からない |

## Botでの実用的な使い方

あなたのTrade[XYZ]バックテストでは、最終的にこう使うのがよいです。

```text
1. 対象銘柄とベンチマークを決める
2. RSラインを計算する
3. RSラインの新高値・新安値を見る
4. 出来高・価格変化・スプレッド・手数料でフィルタする
5. 候補銘柄をランキングする
6. 別の売買ルールでentry/exitする
```

つまり、RSラインは **発注ボタンではなく、銘柄選別フィルタ** です。

## 重要な注意点

RSラインには罠があります。

### 1. 下落相場でもRSは上がる

例：

```text
対象銘柄 -2%
ベンチマーク -5%
```

この場合、対象銘柄は下がっていますが、ベンチマークよりは強いのでRSは上がります。

ロング戦略なら、RSだけでなく、

```text
対象銘柄の絶対リターンもプラスか
```

を見るべきです。

---

### 2. 荒い銘柄ほどRS新高値になりやすい

低流動性銘柄は、単発の跳ねでRS新高値になります。

そのため、

```text
RS新高値
AND 出来高増加
AND スプレッド許容
AND ボラ過大でない
```

のように条件を重ねる必要があります。

---

### 3. ベンチマーク選びで意味が変わる

同じ銘柄でも、何と比べるかで解釈が変わります。

```text
NVDA / SP500
```

なら、米国大型株全体に対する強さ。

```text
NVDA / XYZ100
```

なら、成長株・テック寄り指数に対する強さ。

```text
SOL / BTC
```

なら、暗号資産内でBTCに対する強さ。

## ひとことで言うと

RSラインは、

```text
この銘柄は、基準より強いのか？
```

を見る道具です。

システムトレードでは、

```text
買うための最終判断
```

ではなく、

```text
見るべき銘柄を絞る
弱い銘柄を除外する
市場レジームを判定する
候補をランキングする
```

ために使うのが安全です。
