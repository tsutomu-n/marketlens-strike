このフォルダーは、`marketlens-strike` の現在地と、90%に近づけるための現実的な次手だけを読むためのコピー置き場です。

## 使い方

- actual cash以前の進捗だけを読むなら `PRE_ACTUAL_CASH_PROGRESS_2026-07-04.md`。
- 次に何をすれば90%に近づくかを見るなら `PROGRESS_TO_90_ROADMAP_2026-07-04.md`。

## 注意

90%は利益保証ではない。production live trading readiness でもない。

このフォルダーの要点は、`ACTUAL_CASH_SOURCE_MISSING` を現実的に解消すること、manual ledger を外部証跡そのものと誤認しないこと、1 event の疎通確認を実務上の利益 evidence と過大評価しないことです。
