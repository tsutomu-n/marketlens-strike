
# sourlodine/Solana-Trading-Bot
![](https://opengraph.githubassets.com/17bd8a590ae36a7243835fb35386499a3bef182a7e58ab4083b764d0be735b33/sourlodine/Solana-Trading-Bot)



## Solanaトレーディングボットの概要
- Solanaトレーディングボットv1は、ソラナブロックチェーン上でのトークンの自動取引を実現するソフトウェアツールです。
- ユーザーが設定したパラメータや戦略に基づいて取引を実行し、リアルタイムで市場状況を監視できます。
- ボットのセットアップには、空のソラナウォレットを作成し、SOLを送金してUSDCまたはWSOLに変換する必要があります。
- [Env](https://ja.wikipedia.org/wiki/Env).sampleファイルを更新して設定し、npm installコマンドで依存関係をインストールし、npm run startコマンドでスクリプトを実行します。

## ボットの設定
- 設定ファイルには、ウォレットのプライベートキー、RPCエンドポイント、WebSocketエンドポイント、コミットメントレベルなどが含まれます。
- ボットの設定には、ログレベル、トークンの購入方法、計算ユニット制限、計算ユニット価格、トランザクション実行者などが含まれます。
- トランザクション実行者には、warpまたはjitoを選択でき、カスタムフィーを設定できます。

## 購入設定
- 購入設定には、USDCまたはWSOLのプールをスナイプする、購入金額、自動購入遅延、最大購入リトライ回数などが含まれます。
- 売却設定には、自動売却の有効化、最大売却リトライ回数、自動売却遅延、価格チェック間隔、価格チェック期間、利益確定率などが含まれます。

## 売却設定
- このBOTは、指定された条件に基づいてトークンを購入し、売却するように設定できます。
- 利益確定は、引用銘柄に基づいて計算されます。STOP_LOSSは、損失を確定するためのパーセンテージです。損失は、引用銘柄に基づいて計算されます。
- SELL_SLIPPAGEは、売却時のスリッページ率を指定します。

## スナイプリスト
- Snipeリストを使用すると、指定されたリストに記載されているトークンのみを購入するようにBOTを設定できます。リストは、snipe-list.txtファイルに記載します。
- SNIPE_LIST_REFRESH_INTERVALは、Snipeリストを更新する間隔をミリ秒単位で指定します。

## フィルタ
- フィルタを使用すると、BOTがトークンを購入する条件を指定できます。FILTER_CHECK_INTERVALは、フィルタをチェックする間隔をミリ秒単位で指定します。FILTER_CHECK_DURATIONは、フィルタを満たすまでの待機時間をミリ秒単位で指定します。
- CONSECUTIVE_FILTER_MATCHESは、フィルタを連続して満たす回数を指定します。

## トークン購入条件
- CHECK_IF_MUTABLEは、メタデータが変更不可のトークンのみを購入するようにBOTを設定します。
- CHECK_IF_SOCIALSは、ソーシャルメディア情報が存在するトークンのみを購入するようにBOTを設定します。
- CHECK_IF_MINT_IS_RENOUNCEDは、ミントが放棄されているトークンのみを購入するようにBOTを設定します。
- CHECK_IF_FREEZABLEは、凍結不可のトークンのみを購入するようにBOTを設定します。
- CHECK_IF_BURNEDは、流動性プールが焼却されているトークンのみを購入するようにBOTを設定します。
- MIN_POOL_SIZEは、プールサイズが指定された金額以上のトークンのみを購入するようにBOTを設定します。
- MAX_POOL_SIZEは、プールサイズが指定された金額以下のトークンのみを購入するようにBOTを設定します。

## Warpトランザクション
- Warpトランザクションを使用すると、トランザクションの実行速度が向上しますが、トランザクションの送信先はホストサービスとなります。ただし、ウォレットのプライベートキーは送信されません。
- トランザクションの送信に失敗した場合、手数料は請求されません。
- Warpトランザクションはデフォルトで無効になっています。
- RPCノードがサポートされていない場合、エラーが発生する可能性があります。この場合、RPCノードを変更する必要があります。

## "No SOL token account found in wallet" エラーの対処法
- Solana取引ボットを使用する際に、ログファイルに「No SOL token account found in wallet」というエラーが表示される場合があります。
- このエラーは、提供されたウォレットにUSDC/WSOLトークンアカウントがないことを示しています。
- この問題を解決するには、DEXでSOLをUSDC/WSOLに交換する必要があります。
- たとえば、SOLをWSOLに交換すると、ウォレットにWSOLトークンアカウントが表示されます。
